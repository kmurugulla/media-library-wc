Media Library with Custom CORS Proxy

This example shows how to configure a custom CORS proxy URL for the Media Library component.

CORS Proxy Configuration

Method 1: HTML Attribute (Static)

<media-library 
  cors-proxy="https://your-cors-proxy.com/"
  storage="indexeddb">
</media-library>

Method 2: JavaScript Property (Dynamic)

Enter your custom CORS proxy URL. The proxy should accept requests with a 'url' query parameter.

Method 3: Constructor (Programmatic)

const mediaLibrary = await MediaLibrary.create({
  corsProxy: 'https://your-cors-proxy.com/',
  storage: 'indexeddb'
});
